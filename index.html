Solace Messaging API for Java

The Solace Java API (also referred to as JCSMP) is a 100% pure Java implementation that delivers high message throughput and reasonably low latency.

Solace Messaging API for JMS

The Solace JMS API uses the standard JMS interface to provide JMS service for Java-based applications through a Solace message router. The Solace JMS API supports JMS version 1.1.

Solace Messaging for Pivotal Cloud Foundry

The Pivotal Cloud Foundry Tile provides Solace Messaging as a service within Pivotal Cloud Foundry. These example applications show how Java applications can easily using Cloud Foundry Service Instances to bind and connect their applications to Solace Message Routers.
